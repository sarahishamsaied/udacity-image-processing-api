# Udacity Image Processing API

##  Tools

Simple Node API for resizing images using [Sharp](https://www.npmjs.com/package/sharp)

Express

TypeScript

Jasmine for testing

## API Format for resizing images

```
- /api/resizeImage?filename=map&width=400&height=400
```

### Start the server

 - npm i
 - npm start

### Testing (Run Jasmine)

```
 - npm run test
```

### Linting

```
 - npm run lint
```

### Prettier

```
 - npm run prettier

```
